## 2. 개발 서버 실행 및 외부 브라우저 접속 안내

### 실행 명령

```bash
export PATH="/home/hs-2/.local/share/pnpm/bin:$PATH"
pnpm dev --hostname 0.0.0.0
```

### 서버 주소 확인

`hostname -I` 결과에서 다음 주소를 확인했다.

- LAN 주소: `192.168.0.13`
- Tailscale 주소: `100.76.147.3`
- `172.18.0.1`, `172.17.0.1`은 Docker 내부 네트워크 주소이므로 일반 외부 브라우저 접속 주소로 사용하지 않음

### 실행 중 발생한 상황

- 기존 Next.js 개발 서버가 PID `2939536`으로 3000번 포트를 사용 중이어서 새 실행이 처음에는 3001번으로 이동한 뒤 중복 개발 서버 오류로 종료됨
- 기존 프로세스 안내에 따라 정리한 후 `pnpm dev --hostname 0.0.0.0`으로 다시 실행
- 최종적으로 다음 상태로 준비됨

```text
Local:   http://localhost:3000
Network: http://0.0.0.0:3000
Ready
```

### 사용 방법 설명

- `http://192.168.0.13:3000` 같은 주소는 서버 Bash 터미널에 입력하는 명령어가 아니라 로컬 컴퓨터의 Chrome 또는 Edge 주소창에 입력해야 함
- 같은 LAN에서는 `http://192.168.0.13:3000` 사용
- Tailscale이 로컬 컴퓨터에도 연결된 경우 `http://100.76.147.3:3000` 사용 가능

## 3. 외부 접속 실패 진단

### 서버 측 확인 결과

- Next.js가 `0.0.0.0:3000`에서 LISTEN 중인 것을 확인
- 서버 내부 요청 결과:

```text
HTTP/1.1 200 OK
X-Powered-By: Next.js
Content-Type: text/html; charset=utf-8
```

- 따라서 Next.js 애플리케이션 자체와 3000번 포트 바인딩은 정상
- 접속 실패 지점은 로컬 PC와 서버 사이의 LAN, 방화벽 또는 Tailscale 경로로 판단
- Tailscale 상태에서는 서버 측 장치만 확인되어 로컬 PC가 동일 Tailnet에 연결되지 않았을 가능성을 확인

### 제안한 우회 접속 방법

1. VS Code `PORTS` 탭에서 3000번 포트를 전달한 뒤 로컬 브라우저에서 `http://localhost:3000` 접속
2. 로컬 PC 터미널에서 SSH 터널 생성

```bash
ssh -L 3000:localhost:3000 hs-2@192.168.0.13
```

이후 로컬 브라우저에서 다음 주소 사용:

```text
http://localhost:3000
```

### 최종 화면 확인

- 로컬 브라우저에서 현재 Newron 홈 화면이 표시되는 것을 사용자 제공 화면 캡처로 확인
- 헤더, 메인 히어로 뉴스, 실시간 인기 뉴스, 카테고리 필터, 뉴스 카드 그리드가 렌더링됨

## 4. 헤더와 본문 사이 여백 조정

### 요청 내용

- 화면 상단 헤더 하단과 히어로/인기 뉴스 영역이 너무 붙어 보여 세로 간격을 넓히길 요청

### 원인

- 고정 헤더 높이는 `h-20`
- 본문은 `mt-20`으로 헤더 높이만큼만 밀려 있었음
- 홈 페이지 첫 섹션의 `mt-stack-lg`가 여백을 담당하고 있었지만, 고정 헤더 경계와 가까워 시각적으로 답답하게 보임

### 변경 파일

#### `app/(main)/layout.tsx`

- 본문 레이아웃에 `pt-8` 추가
- 헤더 아래 모든 메인 콘텐츠에 일관된 32px 상단 여백 적용

```tsx
<main className="mt-20 pt-8 max-w-screen-2xl mx-auto px-container-margin pb-16 md:pb-20">
```

#### `app/(main)/page.tsx`

- 첫 섹션에 있던 `mt-stack-lg` 제거
- 상단 여백 책임을 페이지 개별 섹션이 아니라 공통 메인 레이아웃으로 이동

```tsx
<section className="grid grid-cols-1 lg:grid-cols-10 gap-gutter mb-stack-lg">
```

### 기대 효과

- 헤더 경계와 히어로/실시간 인기 뉴스 제목 사이가 32px 벌어짐
- 메인 페이지 상단 간격이 한 위치에서 관리되어 이후 페이지에서도 일관된 레이아웃 유지 가능
