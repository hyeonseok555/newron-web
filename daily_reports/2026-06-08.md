## 5. 참고 사항

- 현재 `.git` 디렉터리는 비어 있는 형태로 보여 `git status`는 정상적인 변경 추적 수단으로 사용할 수 없다.
- 개발 서버 실행 중 Slow filesystem 경고가 표시되었다. `/hdd4/newron-web/.next/dev` 경로의 파일시스템 응답이 느릴 수 있다는 의미이며, 즉시 차단 이슈는 아니다.
- 사용자가 터미널에서 확인한 실행 이슈를 문서에 반영했다.
  - `rt PATH=...`는 오타이며 `export PATH=...`를 사용해야 함
  - `http://localhost:3000`은 터미널 명령어가 아니라 브라우저 주소
  - `Another next dev server is already running`은 기존 서버가 이미 실행 중이라는 뜻이며, 헬스체크가 정상이면 그대로 사용 가능
- SSH 끊김 원인 확인 중 `enp6s0`의 링크 다운/업 반복 및 `100Mbps` downshift 로그를 확인했다.
- 사용자가 듀얼 랜 보드의 다른 포트로 변경한 뒤 `enp7s0: Link is Up - 1Gbps/Full` 로그를 확인했다.
- 실시간 네트워크 링크 감지 스크립트를 추가했다.
  - `scripts/watch-network-link.sh`
  - 출력 로그: `logs/network-link-watch.log`
- 사용자가 바로 읽기 쉽도록 네트워크 링크 감지 스크립트의 시작 메시지와 README 설명을 한국어로 정리했다.
- 다음 구현 작업은 기획서 기준상 홈 피드의 정적 목업을 실제 `/news`, `/categories` API 기반으로 교체하는 것이 자연스럽다.
